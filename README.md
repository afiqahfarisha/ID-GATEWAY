# ID-GATEWAY
# Final Year Project
# Overview
ID-GATEWAY is a network gateway management system developed as a Final Year Project (FYP). The system is designed to simplify network administration, improve monitoring capabilities, and provide secure access management through a centralized platform.
# Objectives
- To develop an efficient gateway management system.
- To enhance network monitoring and administration.
- To improve security and accessibility for network users.
# Features
- User Authentication
- Gateway Access Control
- Network Monitoring Dashboard
- Report Management
- User-Friendly Interface
# Technologies
- HTML
- CSS
- JavaScript
- PHP
- MySQL
